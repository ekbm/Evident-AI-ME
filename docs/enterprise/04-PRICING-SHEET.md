# Evident — Enterprise Pricing

**Effective: March 2026**

---

## Pricing Philosophy

Simple, transparent, and fair. Pay a low base price for platform access, and only pay more when your team uses more AI features. No hidden fees, no per-feature charges, no surprises.

---

## Platform Tiers

### Evident Cloud — $2.99/user/month
Everything you need to get started immediately.

**Includes:**
- Full platform access (Chat with Evi, Knowledge Space, Knowledge Health)
- All AI features (Q&A, obligation extraction, document analysis, OCR, transcription)
- All file types supported (PDF, Word, Excel, images, audio, video)
- External integrations (SharePoint, Google Drive, Confluence, Slack, CRM)
- Document readiness scoring and AI preparation pipeline
- Admin console and user management
- 100 free AI queries per user per month
- Email support
- Automatic updates

**Best for:** Teams wanting fast setup with no IT overhead.

---

### Custom Storage — $7.99/user/month
Everything in Evident Cloud, plus your own storage.

**Additional benefits:**
- Store documents in your own cloud bucket (AWS S3, Azure Blob, or Google Cloud Storage)
- Choose your data region for residency compliance
- Self-service bucket configuration via admin console
- Evident processes and indexes your documents, but files stay in your storage

**Best for:** Organisations with data residency requirements or existing cloud storage.

---

### Private Cloud — $14.99/user/month
Minimum 100 users. Everything in Custom Storage, plus dedicated infrastructure.

**Additional benefits:**
- Dedicated Evident instance deployed in your cloud account (AWS, Azure, or GCP)
- Full network isolation (your own VPC)
- Dedicated database (not shared)
- Customer-managed encryption keys (CMEK)
- Remote management and updates by Evident
- Priority support with SLA included

**Best for:** Organisations needing complete cloud isolation and dedicated resources.

---

### Full On-Premise — Quote-based
Minimum 250 users. Everything runs on your servers.

**Additional benefits:**
- Containerised deployment (Docker/Kubernetes) on your infrastructure
- All data stays within your network — nothing leaves your premises
- Optional local AI models (no external API calls) for air-gapped environments
- Quarterly update packages
- On-site or remote installation support
- Dedicated account manager

**Best for:** Government, defence, banking, healthcare, and highly regulated industries.

## Usage-Based AI Pricing

All tiers include **100 free AI queries per user per month**. After that:

| Action | Cost per unit |
|--------|--------------|
| AI Q&A query | $0.015 |
| Document processing (OCR, text extraction, prep) | $0.02 |
| Deep document scan (readiness assessment) | $0.03 |
| CV / Report generation | $0.05 |

**Example:** A team of 50 users where each user asks an average of 150 queries/month:
- Base cost: 50 x $2.99 = $149.50/month
- Free queries: 50 x 100 = 5,000 queries (no charge)
- Overage: 50 x 50 = 2,500 queries x $0.015 = $37.50/month
- **Total: $187.00/month** ($3.74/user effective)
