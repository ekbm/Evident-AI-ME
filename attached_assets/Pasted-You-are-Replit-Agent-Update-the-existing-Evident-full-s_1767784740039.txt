SCORING MODEL (IMPLEMENT EXACTLY)
Compute readiness score from last 30 days data, then map to 0..100.

1) COVERAGE (30 points)
- uploaded_assets_count = COUNT(assets)
- target_assets_count = configurable setting (default 30), stored in settings table as key="target_assets_count"
- coverage_ratio = min(1.0, uploaded_assets_count / target_assets_count)
- missing_context_count_last30d = COUNT(readiness_events where type="missing_context" in last 30 days)
- missing_context_penalty = min(10, missing_context_count_last30d)
- coverage_score = 30 * coverage_ratio - missing_context_penalty
Clamp 0..30

2) STRUCTURE (20 points)
- structured_exports_count_last30d = COUNT(readiness_events where type IN ("export") last 30 days) +
  COUNT(actions run like "extract_obligations" stored as readiness_events type="export" or type="structure_run"
- structured_assets_ratio = structured_exports_count_last30d / max(1, uploaded_assets_count)
- structure_score = min(20, 20 * structured_assets_ratio)
Clamp 0..20

3) RETRIEVAL QUALITY (20 points)
- total_questions_last30d = COUNT(readiness_events where type in ("chat_answered","chat_not_found") last 30 days)
- answered_questions_last30d = COUNT(type="chat_answered" last 30 days)
- answered_ratio = answered_questions_last30d / max(1, total_questions_last30d)
- retrieval_score = 20 * answered_ratio
Clamp 0..20
Definition:
- “chat_answered” if response includes citations and does NOT contain "Not found"
- “chat_not_found” if response indicates missing evidence / not found.

4) FRESHNESS & GOVERNANCE (15 points)
- assets_with_metadata = COUNT(artifacts where metadata_json not empty grouped by asset)
- freshness_ratio = assets_with_metadata / max(1, uploaded_assets_count)
- conflicts_penalty = min(5, COUNT(readiness_events where type="conflict_detected" last 30d))
- freshness_score = 15 * freshness_ratio - conflicts_penalty
Clamp 0..15

5) ADOPTION & WORKFLOW (15 points)
- reports_enabled = reports.enabled==1 (weekly or monthly)
- report_run_count_last30d = COUNT(readiness_events where type="report_run" last 30d)
- usage_days_last30d = distinct days where any readiness_event occurred last 30d
- adoption_score starts at 0:
  +5 if reports_enabled
  +5 if report_run_count_last30d >= 1
  +5 if usage_days_last30d >= 4
Clamp 0..15

TOTAL
score_total = sum component scores, clamp 0..100.

CONFIDENCE LEVEL
