You are Replit Agent. Upgrade the existing “Evident” app to support TWO MODES focused entirely on INTERNAL DATA ONLY (no external/web references).

MODE 1: INDIVIDUAL — “Knowledge & Reports from Your Files”
- Users upload offline files (documents, images, audio, video).
- Evident extracts structured data, answers questions with citations, and generates reports.
- All answers must come strictly from the user’s uploaded files.
- If information is missing, Evident must say “Not found in provided files.”

MODE 2: ENTERPRISE — “AI Readiness from Existing Systems”
- Organisations connect offline or internal data sources (folders, exports, drives).
- Evident ingests data on a daily snapshot basis (read-only).
- Evident measures progress toward AI readiness using internal data only.
- No external sources, no web search, no blending.

GOAL
Prepare Evident to be a safe, trustworthy pathway for individuals and organisations to:
- Understand existing knowledge
- Structure it for reuse
- Generate reports
- Progress toward AI readiness using their OWN data

CORE PRINCIPLES (MUST FOLLOW)
- Internal evidence ONLY.
- Every answer must include citations to uploaded files OR clearly say “Not found”.
- No guessing, no augmentation from the internet.
- Read-only by default.
- Human review encouraged everywhere.

- Backend: Node.js + Express (existing)
- DB: SQLite
- Files: local /uploads for MVP (S3 later)
- AI: embeddings + responses + transcription/vision for uploaded media only

2) Local Folder Connector (MVP-friendly)
- Config: local server folder path
- Daily snapshot scan (hash/mtime)
- Ingest new/changed files

(Optional later: S3 Drop Connector)

D) Scheduler

- Execute:

  - Weekly/monthly reports

  - Contract obligations
